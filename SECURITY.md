# Security

AI Chat Vault is local-only. It does not send conversation content to a server, does not request extension storage permissions, and does not include analytics.

## Reporting

Please report security issues privately to the repository owner. Include reproduction steps, browser version, and the affected extension version when possible.

## Scope

Supported hosts are limited to:

- `https://chatgpt.com/*`
- `https://chat.openai.com/*`

Issues involving third-party builds, modified source, or unrelated websites are out of scope.
